In Tarossary you can create your own glossary and links will automatically be attached to articles

== Description ==

You can create your own glossary at Tarossary and the link will be automatically attached to the article.
You can also treat custom posts as terms.

== Installation ==

1. Upload the `tarossary` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the Plugins menu in WordPress.
